# fall-detection — 老人跌倒识别摄像头

基于计算机视觉的跌倒识别项目，用于家中老人看护。方向：光电 + AI 交叉。

## 技术栈

- Python
- OpenCV（摄像头与图像处理）
- MediaPipe（姿态估计，后续阶段引入）
- 运行在带独显的笔记本上；成品阶段可移植到树莓派 / Jetson

## 运行方式

```
# 1. 创建并激活虚拟环境
python -m venv .venv
.venv\Scripts\activate          # Windows PowerShell

# 2. 安装依赖
pip install opencv-python

# 3. 运行
python 打开摄像头.py
```

窗口显示摄像头实时画面，按 q 退出。
